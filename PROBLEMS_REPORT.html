1. Lỗi IDE (Java Language Server cache) — 3 lỗi

⚠ Đây là lỗi giả do Java LS chưa re-index sau khi thêm file mới. Backend vẫn build bình thường bằng Maven. Cách fix: Ctrl+Shift+P → "Java: Clean Java Language Server Workspace".

Cảnh báo của IDE khi dùng pattern .orElseThrow(). Code hoàn toàn an toàn ở runtime. Không cần sửa — đây là pattern chuẩn của Spring Boot.
